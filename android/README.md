# Grünstahl — Android App

Native Android app for reading and writing the Grünstahl FRAM device over USB HID.
Features a hex editor and text editor matching the web version.

---

## Prerequisites

### Build machine
- **JDK 17+** — `java -version` to check
- **Android SDK with Build Tools 34** — install via Android Studio, or:
  ```bash
  # Download command-line tools from:
  # https://developer.android.com/tools/sdkmanager
  sdkmanager "build-tools;34.0.0" "platforms;android-34"
  ```
- Set `ANDROID_HOME` to your SDK location:
  ```bash
  export ANDROID_HOME=$HOME/Android/Sdk        # Android Studio on Linux
  export ANDROID_HOME=/usr/lib/android-sdk     # apt install android-sdk
  export ANDROID_HOME=$HOME/Library/Android/sdk # macOS
  ```
- **curl** or **wget** (to download Gradle on first run)

### Phone
- Android 8.0 (API 26) or later
- USB-C OTG cable or adapter

---

## Building

`build.sh` downloads Gradle 8.4 automatically on first run (~100 MB, cached in `~/.gradle`).

```bash
cd android/

# Build debug APK
./build.sh

# Build release APK (debug-signed, ready to sideload)
./build.sh release

# Build debug APK + install directly via adb
./build.sh install
```

Output:
- Debug:   `app/build/outputs/apk/debug/app-debug.apk`
- Release: `app/build/outputs/apk/release/app-release.apk`

---

## Installing (sideloading)

### Via adb (easiest)
```bash
# Enable on phone: Settings → Developer Options → USB Debugging
./build.sh install
# or manually:
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

To enable Developer Options: Settings → About Phone → tap **Build Number** 7 times.

### Manual sideload (no PC needed after APK is on phone)
1. `./build.sh release`
2. Copy `app-release.apk` to your phone (USB, cloud, email, etc.)
3. On phone: Settings → Security → **Install unknown apps** → allow your file manager
4. Open the APK in the file manager → tap **Install**

---

## Connecting Grünstahl

1. Plug Grünstahl into your phone via USB-C OTG
2. The app launches automatically and asks for USB permission
3. Tap **OK** (tick "Always allow" to skip next time)
4. FRAM is read automatically on connect

```
├── build.gradle                      Root Gradle config
├── settings.gradle

    ├── build.gradle

        ├── AndroidManifest.xml

        │   ├── layout/               activity_main.xml, dialog_edit_byte.xml
        │   ├── values/               strings, themes, colors
        │   ├── drawable/             shapes, cursor, icons
        │   └── xml/usb_device_filter.xml
        └── (generated by build: databinding, R.java)
```
